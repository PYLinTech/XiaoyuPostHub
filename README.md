# XiaoyuPostHub

> 自建云盘与资源协作管理系统

XiaoyuPostHub 是面向个人、团队和组织的自建云盘与资源协作管理系统，提供完整的文件存储、分享、协作与治理能力，支持私有化部署。

## 核心特性

- **私有化部署**：支持本地、局域网、内网或专有云环境部署，数据完全自主可控
- **权限可控**：基于用户、角色与组织结构的细粒度权限模型，访问范围可按需收敛
- **稳定的上传下载**：支持大文件分片、断点续传、秒传与并发限速
- **分享可治理**：链接可设置有效期、密码、访问次数与水印，过期自动失效
- **审计可追溯**：操作日志完整记录，文件变更、权限调整、分享行为均可回溯
- **容量可计量**：按用户、团队、组织维度统计容量使用情况，支持限额与预警
